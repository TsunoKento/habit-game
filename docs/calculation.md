# 計算ロジック

## 総合経験値

全習慣の達成記録から動的に算出する。DBには保存しない。

```
Total EXP = SUM(達成回数_i × exp_per_done_i)
```

各習慣の `exp_per_done` の合計は 100 とする。これにより、全習慣を1日すべて達成すると 100 EXP を獲得する。

---

## 現在レベル

総合経験値から以下の式で算出する。

```
Level = floor(totalEXP / 100) + 1
```

レベルアップに必要な経験値は一律 100 EXP。全習慣を1日すべて達成すると1レベル上がる。

---

## 連続達成日数

対象習慣について、過去に向かって日付が連続してレコードが存在する日数を数える。

1. 起点は「当日のレコードがあれば当日、なければ前日」とする
2. 起点から1日ずつ遡り、レコードが存在する限りカウントを増やす
3. レコードが存在しない日（空白日）が現れた時点でカウントを停止する
4. 起点にレコードが存在しない場合は 0 を返す
